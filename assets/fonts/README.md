### Download links:
1. **PressStart2P-Regular.ttf**: https://github.com/google/fonts/raw/main/ofl/pressstart2p/PressStart2P-Regular.ttf
2. **Montserrat-Bold.ttf**: https://fonts.google.com/download?family=Montserrat (select Bold weight)

### Or via Google Fonts API:
```bash
# PressStart2P
curl -o PressStart2P-Regular.ttf "https://fonts.gstatic.com/s/pressstart2p/v9/NMif7zDgSBIc5K1SlfKT8PgLQwE.ttf"

# Montserrat Bold (700 weight)
curl -o Montserrat-Bold.ttf "https://fonts.gstatic.com/s/montserrat/v26/JTUHjIg1_i6t8kCJq5nH61-brhTWotXiUXZrNA.ttf"
```
